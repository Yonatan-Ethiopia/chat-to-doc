# ChatDoc 📖  
---

**ChatDoc** is a project to turn AI chats into structured documents like EPUBs and PDFs(currently only Epubs are supported).  

---  
## Features 

* **Turn messy chat html into structured epub files**  
* **Turn each user's questions into chapters, sections or subsections based on how related the topics are**  
* **Generate a short title for each chapter or section from the user's questions to the AI chatbot**  
* **Sequential Job Worker :** Implements a polling-based worker loop that monitors for new tasks, processes them sequentially to ensure data integrity, and returns to a wait state when idle.
